# 🍽️ Receitas

Este projeto foi desenvolvido com o objetivo de praticar a construção de páginas web utilizando apenas **HTML** e **CSS**, focando na estruturação de conteúdo e estilização de layouts para exibição de receitas.

## 🚀 Conteúdos abordados

Durante o desenvolvimento deste projeto, foram trabalhados os seguintes conceitos:

- Estruturação de páginas com HTML5  
- Uso de tags semânticas  
- Estilização com CSS3  
- Organização de layout (cores, espaçamento, tipografia)

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  

## 📦 Como rodar o projeto

```bash
# Clone o repositório
git clone <url-do-repositorio>

# Acesse a pasta
cd receitas

# Abra o arquivo no navegador
Basta abrir o index.html no seu navegador
```
